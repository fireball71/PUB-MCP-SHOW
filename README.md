# MCParks Show Tracker
> Track live show times in-game, displayed in your local time zone.

**MCParks Show Tracker** is a Fabric client mod built for the [MCParks](https://mcparks.us/) Minecraft server. It brings real-time show schedules directly into your HUD — countdowns, notifications, and one-click warping, all without leaving the game.

---

## ✨ Features

- 🕐 **Live show countdowns** displayed in your local time zone
- 🔔 **Notifications** when a show is about to start
- 🚀 **Warp button** to teleport directly to a show's location
- 🪄 **MagicBands integration** — right-click the show button in MagicBands to open the show menu

---

## 📋 Requirements

- Minecraft **1.18.2**
- [Fabric Loader](https://fabricmc.net/)
- [MCParks Server](https://mcparks.us/)

---

## 🔧 Installation

1. Install [Fabric](https://fabricmc.net/use/installer/) for Minecraft **1.18.2**
2. Download the latest release of PUB-MCP-SHOW
3. Drop the `.jar` file into your `.minecraft/mods` folder
4. Launch Minecraft and connect to MCParks
5. Open your MagicBand and **right-click** the show button, or press **F8**
